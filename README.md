A virtual terminal ```ls``` that color-codes projects by importance.

| Color	      | Description		|
| ----------- | ---------------	|
| Red	      | Super important	|
| Green       | Active			|
| Yellow	  | Updateable		|
| White		  | Not important	|

To config a colored directory:

